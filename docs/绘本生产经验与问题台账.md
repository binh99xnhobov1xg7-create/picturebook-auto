## 1. 问题台账（问题 → 现象 → 根因 → 修复 → 涉及文件/函数）

### 1.1 提示词被 4000 截断 → 主角退化呆站 / 跳题

- **现象**：出图主角并排"摆拍"、不演本页情节、跳题；IP 漂移、克隆、比例错频发；画面不可控。
- **根因**：API 端把最终 prompt **硬截到 4000 字符**（含首尾英文画风段）。原 `GPT_CLEAN_STYLE_DIRECTIVE/ECHO` 合计约 **3900 字**，几乎把中文正文（场景/IP/比例/防分身锁）全部挤掉——实测中文正文仅剩约 **209 字**存活，尾部"画风/留白/防分身"被悄悄切掉。这是"出图不可控 / IP 漂移 / 克隆 / 比例错"的**真正根因**。
- **修复**：
  1. 英文画风首尾两段大幅压缩（DIRECTIVE ~430、ECHO ~850），只保留画风核心 + 禁文字锁 + 留白锁，把宝贵额度让给中文关键铁律；
  2. 中文正文瘦身、去重，并把 **IP 锁 / 配色锁上移**（紧跟开头），确保不被尾截；
  3. 设软上限 `PROMPT_SOFT_LIMIT=3800`，`enforce_prompt_budget()` 去重并对超限页打 `[prompt-budget]` 告警。
- **涉及**：`config.py`（`_SOP_STYLE_DIRECTIVE/ECHO`、`_NO_TEXT_AND_LAYOUT_LOCK`、`PROMPT_SOFT_LIMIT`、`dedupe_prompt_sentences()`、`enforce_prompt_budget()`）；`cn_prompt_builder.py`（构造顺序：核心画面/IP/配色锁上移，见注释"把额度让给比例/防分身/国际化"）。
- **复用要点**：**每页 dry-run 必须 < 3800（硬不超 4000）**；任何新增锁都要算预算，宁可精简不可让尾部铁律被截。

### 1.2 代词页 Tommy 深藏青回退、Mia 散发回退

- **现象**：只用代词（he/she/they，无显式人名）的页里，Tommy 被画成深藏青 polo+牛仔、Mia 散发——回退到旧形象。集中出现在 Book69 P3/P4/P6。
- **根因**：这些代词页缺少显式主角名挂钩，兜底走了**过期的 `trio_style_anchor.png`**，把旧定妆（深藏青/散发）带了回来。
- **修复**：
  1. **重建 trio 兜底图**（`_rebuild_trio_anchor.py`），使其与现行定妆一致；
  2. 代词页**强制挂主角【按级别年龄】的新定妆锁**：Tommy 浅天蓝长袖卫衣、Mia 脑后中高马尾+紫发圈、10 岁。
- **涉及**：`character_registry.py`（Tommy/Mia 10 岁定妆 key：`light blue LONG-SLEEVE sweatshirt, khaki straight pants, white sneakers` / `dark-brown hair gathered entirely into ONE high ponytail`）；`cn_prompt_builder.py`（`_lead_keys_for_book` 代词页主角判定与注入）；`references/official_style/trio_style_anchor.png`。

### 1.3 主题锁污染（示例词污染 `_THEME_SIGNATURE` 误判主题）

- **现象**：警察 / 消防 / 医生页硬贴上花盆、铲子、浇水等种植道具（Book66 根因）。
- **根因**：非虚构注入逻辑会往 `scene_cn` 追加 `(一起播种/采摘/清洗/搬运/挑选等)` 这类**示例动作词**，其中"播种"含"种"被 `_THEME_SIGNATURE` 的"种植/园艺"正则（`种|garden|plant…`）误命中，把园艺道具错贴到别的主题页。
- **修复**：在注入这些示例词**之前**，用原始 `scene_cn` 快照一份 `_raw_scene_for_theme`，专供 `_scene_story_lock()` 做主题判定；并优先按**本页场景**匹配（图书馆页→书架、博物馆页→展柜），无明确主题词时才回退全书主题/标题。
- **涉及**：`cn_prompt_builder.py`（`_THEME_SIGNATURE`、`_scene_story_lock()`、`_raw_scene_for_theme` 快照，约 1898-1902 行注释）。

### 1.4 职业复数误判（firefighters 被当一次性小孩）

- **现象**：`firefighters / Officers / Doctors` 等复数大写职业词被画成 10 岁儿童（Book66 根因）。
- **根因**：职业名只匹配了单数 / 受 `\b` 词界影响，**复数大写词漏过成人判定**，被后面"单词专名→儿童"兜底误锁成 10 岁儿童。
- **修复**：`_ADULT_ROLE_RE` **同时匹配单复数**（`officers?|firefighters?|firemen|doctors?|nurses?|farmers?…`）并对动物/职业**成人优先**判定；成人角色锚图加"成熟成年人脸 + 成人身材比例、约为儿童身高 4:3、7-7.5 头身"锁。
- **涉及**：`book_cast.py`（`_ADULT_ROLE_RE`、`is_adult_role()`、`_ANIMATE`，约 213-233 行）。

### 1.5 "围观人群"负向被误判为 extras

- **现象**：纯双主角内页错误触发"配角确定性配色轮 + 配角多元各异 + 封面/群像等高锁"整段噪声（修 2026-06-10）。
- **根因**：非虚构/连续性注入会往 `scene_cn` 追加"**不要凭空添加陌生路人/围观人群…**"等**负向指令文案**，其中"围观人群/路人"被 `_scene_has_nonlead_extras()` 的人群正则**误判成"本页真有配角"**。
- **修复**：在**注入负向样板之前**对未被污染的真实场景快照 `has_extras`（`_scene_extras_raw`），再传给 `_build_positive_concise()`；负向构造方接收外部快照值，不再二次扫描被污染的文案。
- **涉及**：`cn_prompt_builder.py`（`_scene_extras_raw` 快照、`_build_positive_concise(..., has_extras=...)`、`_scene_has_nonlead_extras()`，约 1892-1897、2247-2250 行）。

### 1.6 色斑 / 磕碎感（放大后处理放大水彩底颗粒）

- **现象**：放大后大片渐变 / 平滑区出现斑驳色斑、割裂"磕碎"感。
- **根因**：Real-ESRGAN 未装时回退 **Lanczos + 全图 USM**，USM 把水彩底颗粒在平滑区放大成色斑。
- **修复**（用户拍板 2026-06-10）：
  1. **优先探测 Real-ESRGAN**（`realesrgan_ncnn_py` NCNN 或 `realesrgan` torch），自带高质量超分、无需 USM；
  2. 不可用时**显著弱化 USM**：`IMAGE_USM_PERCENT 45→22`、`RADIUS 1.2→1.0`、`THRESHOLD 3→6`，且默认**仅对边缘锐化**（`IMAGE_USM_EDGE_ONLY=true`，平滑/渐变区不锐化）；
  3. 可选对平滑区**轻度降噪**（`IMAGE_FLAT_DENOISE=true`，`RADIUS 0.7`）消除斑驳、保留均匀纸纹。
  - 回退旧式全图锐化：`IMAGE_USM_EDGE_ONLY=false`。
- **涉及**：`config.py`（`IMAGE_UPSCALE_METHOD`、`IMAGE_USM_*`、`IMAGE_FLAT_DENOISE*`，约 324-337 行）；`seedream_client.py`（`_upscale_esrgan()`、放大主流程，约 96-176 行）。

### 1.7 画风迭代史（软水彩 → 墨线+莫兰迪 → 轻水彩 → 清透明亮纯净 E 定稿）

- **演进**：
  - **A 软水彩**（旧默认，`_WATERCOLOR_STYLE_DIRECTIVE/ECHO`）；
  - **B 经典墨线 + 纹理纸 + 莫兰迪**（SOP 原文画风，`_SOP_STYLE_DIRECTIVE/ECHO`，`SOP_STYLE_MODE=true` 启用）；
  - **C/D 轻水彩过渡**（`_style_c.py / _style_d.py` 校准，去莫兰迪）；
  - **E 清透明亮纯净定稿**（`_style_e.py`）：**彻底去掉莫兰迪/muted/dusty/grey**，改"CLEAR TRANSPARENT LIGHT WATERCOLOR—pure luminous colors、thin translucent washes、white paper glowing through、细墨线、light-to-medium 干净不灰"。
- **用户反复诉求（务必记牢）**：**通透 / 明亮 / 纯净 / 干净不脏不灰、不发闷不发厚、绝不莫兰迪/dusty/grey/muddy/dark、绝不 neon**；人物**低龄可爱**、绝不写实大龄；脸颊**至多极淡自然红晕**，禁强烈红/粉腮红圈。
- **现状**：默认 `SOP_STYLE_MODE=true`；两套画风共用同一份【禁文字锁 + 留白锁】，IP 锁由 cast 注入、两套通用。
- **涉及**：`config.py`（`SOP_STYLE_MODE`、`GPT_STYLE_*`、`_SOP_STYLE_*`、`_WATERCOLOR_STYLE_*`，约 92-218 行）。

### 1.8 IP 锁要点（系列主角与一次性/动物角色）

- **Mia（10 岁）**：脑后**中高马尾**、紫色发圈、紫色系上衣；HAIR LOCK 强调"全部头发束成一根马尾，NOT 丸子头 / NOT half-up / NOT 散发 / NOT 低马尾侧马尾"。
- **Tommy（10 岁）**：**浅天蓝长袖圆领卫衣** + 卡其直筒裤 + 白鞋；短乱棕发；NOT a girl / NO ponytail / NO long hair。
- **一次性外籍小孩**：**不准参考主角锚图**、不撞脸、**不穿蓝/紫**；走国际化多元长相，与主角长相完全不同。
- **警察 / 农夫 / 工人等职业角色**：**必须成人**（成熟成人脸 + 成人身材，约儿童身高 4:3、7-7.5 头身）；绝不画成小孩。
- **剧情狗 Buddy（Book63 走失小狗）**：全书唯一形象锁——杏奶油色蓬松波浪卷毛混血、膝盖高、圆脸黑鼻、**半垂耳**（非柯基立耳）、**鲜明红色项圈 + 小圆金吊牌**；跨页一致（品种/毛色/项圈）；**严禁画成家养柯基 Max**（Max 仅在正文显式点名时出现）。
- **涉及**：`character_registry.py`（主角与配角定妆）；`book_cast.py`（`STORY_DOG_LOCK_CN`、`is_story_dog_role()`、`is_animal_role()`、成人锁）。

### 1.9 配角 / extras 规则 与"酷似主角海报脸"

- **配角量化**（SOP 第 6 条）：生活化配角总数 **≤2**、视觉占比 **≤15%**、仅背景点缀、不抢主角；永久禁 Dino。
- **反分身（主角铁律 §3）**：每页同一角色只出现一次；背景人群只能远景虚化的、与主角明显不同的**成人**；**严禁背景出现紫色高马尾女孩 / 蓝衣蓬松短发男孩等酷似 Mia/Tommy 的小孩**（会变分身）。成人工人/农夫**绝不能画成与主角同款长相**。
- **装饰性小动物 / "酷似主角海报脸"**：刚决定加锁的复发问题——见 §4 待观察。
- **涉及**：`cn_prompt_builder.py`（负向构造、`has_extras` 分支）；`seedream_client.py` 自审第④项（分身/超员）。

### 1.10 命名角色"场景描述误当形象注入"（Book63 根因）

- **现象**：图文不符、每页复述压过本页剧情、顶部留白变生硬色块。
- **根因**：`book_cast` 把命名角色的**场景描述**误当成"角色外观"注入，且每页复述。
- **修复**：区分"角色外观锁"与"本页场景动作"；顶部留白改为**场景自然延展**（非色块）。
- **涉及**：`book_cast.py`、`cn_prompt_builder.py`。

## 2. 复用清单 / 跑新书 checklist

### 2.1 出图前 dry-run 必验

- [ ] **每页 prompt < 3800 字符**（硬不超 4000）；看 `[prompt-budget]` 有无告警。
- [ ] **★0 核心画面 + IP 锁（Mia 马尾紫 / Tommy 浅天蓝 / 按级别年龄）完整保留**、未被尾截。
- [ ] 负向与正向**无自相矛盾**（如"不要围观人群"没被误判成有配角）。
- [ ] 主题锁命中**本页真实场景**（不串题）；职业角色判成**成人**；动物专名逐字保留。
- [ ] 框架式寓言走方案 B（读者缩左下角 + 幻象铺满）；连续天气贯穿。

### 2.2 出图后逐页自查 7 项（硬伤门）

1. **图文相符**：画对本页关键道具/动作/场景，没跳题、没漏关键情节。
2. **主角亲手参与**：Mia/Tommy 是实施者（动手就操作、观看就专注看/指认），不被晾在边缘当路人。
3. **标志场景/道具**：非虚构主题页出现该主题标志物（书架/展柜/垃圾扫帚/狗/手机照片…）。
4. **清透画风无色斑**：通透明亮纯净、不脏不灰、无斑驳磕碎、无硬色块/调色板感。
5. **IP 正确**：Mia=中高马尾+紫；Tommy=浅天蓝长袖；10 岁等高同龄；成人明显高于孩子；狗=Buddy 锁。
6. **无克隆分身**：每页女孩≤1、男孩≤1，背景无酷似主角的小孩。
7. **无多余路人**：默认画面只有 Mia 和 Tommy（+ 必要的 1 位具名成人），删多余陌生人；无可读英文。

> 自审第⑦项（剧情/场景匹配）已固化进 `seedream_client.py` 的 GPT 视觉自审（2026-06-10 新增），仅审出硬伤才**定向修图**（图生图保留构图/画风），无问题原样交付。

### 2.3 并行批量节奏（实测最优）

- **4 本一批并行**：实测四本几乎完全同时（~18 分钟跑完，墙钟），相当于串行 ~60 分钟的 1/3，**gpt-image-2 全程零限流/零 429**。
- 每本各自**独立进程 + 独立输出目录**（内存隔离、IP/cast 零交叉污染）。
- 经验上限：**同时 ≤ 2-3 个书进程更稳**；若与其它正在跑的书共享同一图像 API 配额，并发过高会触发 429（撞限流会退避/停，反而拖慢、可能丢页）。四本"零限流"是在独占配额时实测得到的，按当时配额放心 4 本/批。
- 遇配额/限流/报错**立即停并回报，绝不反复重试烧钱**。

### 2.4 非破坏性备份 与 WPS/Office 文件锁绕过

- **备份**：重出前对要替换的资产做 `.bak` 备份（如 `*.bak.png`），删旧 `images/`（含 `_anchors`）前确认已备份；临时脚本保留为未跟踪文件即可。
- **文件锁**（本机用 **WPS** 不是 Office）：`Reader.pptx` 被 WPS 占用会导致 `PermissionError`。释放：
  ```powershell
  Get-Process POWERPNT,wpp,EXCEL -ErrorAction SilentlyContinue | Stop-Process -Force
  ```
  若用户正在 WPS 里看另一本，**不要盲杀进程**（怕丢未保存内容）——先请用户手动关闭目标本，再重跑。
- **PowerShell 注意**：不要用 `&&` 串命令；python 用 `py`。

### 2.5 成本估算

- 整本重跑（7 页 + 封面）≈ **¥20~25/本**；4 本并行 ≈ **¥80~95**。
- 定向补修：2 页 ≈ **¥6~8**，单页 ≈ **¥3**。
- 单本预算硬约束 ~¥25；超预算/报错即停。

## 3. 稳定生图参数（可复用配置 · 现值）

| 参数 | 现值 | 说明 |
|---|---|---|
| `IMAGE_QUALITY` | `high` | gpt-image 系列 quality（此前未设走默认档偏软，设 high 显著提清晰度） |
| `IMAGE_SIZE` | `1536x1024` | gpt-image-2 不支持 4:3 直出；先出 3:2 → 居中裁 4:3 → 升分辨率 |
| `IMAGE_TARGET_PRINT` | `2000x1500` | 工作图（进 PPT）4:3 |
| `IMAGE_PRINT_DELIVER_PX` | `3000x2250` @ `300` DPI | A5 印刷交付（CMYK/TIFF/PDF 可选） |
| `IMAGE_UPSCALE_METHOD` | `lanczos`（优先探测 ESRGAN） | 非显式 lanczos 时先试 Real-ESRGAN |
| `IMAGE_USM_PERCENT / RADIUS / THRESHOLD` | `22 / 1.0 / 6` | 弱化 USM，防色斑（旧值 45/1.2/3） |
| `IMAGE_USM_EDGE_ONLY` | `true` | 仅边缘锐化，平滑区不动 |
| `IMAGE_FLAT_DENOISE` / `_RADIUS` | `true` / `0.7` | 平滑区轻度降噪 |
| `PROMPT_SOFT_LIMIT` | `3800` | 软上限；超限去重 + 告警（防 4000 截断） |
| `CONCISE_PROMPT` | `true` | 简短正向提示词（实测比"几百条负向墙"更干净） |
| `SOP_STYLE_MODE` | `true` | 默认 SOP 画风；回退旧水彩设 false |
| `IMAGE_SELF_REVIEW` | `true` | GPT 视觉自审，仅硬伤才定向修图 |
| `VISION_REVIEW_MODEL` | `claude-sonnet-4-6` | 自审多模态模型（本账号 gpt/gemini 无渠道，用 Claude 系） |
| `IMAGE_POLL_MAX_TRIES` | `180` | 慢任务给 ~15 分钟再判失败（gpt-image-2 偶发超时，失败自动补跑） |
| `LEVEL_IMAGE_BACKEND` | 全级别 `gpt` | A/B 对比后选定纯 gpt-image-2（画风更干净、遵循度高） |

## 4. 遗留 / 待观察

1. **装饰性小动物 / "酷似主角海报脸"复现率**：背景小松鼠/小猫/小鸟等装饰性小动物，以及背景偶发"酷似主角的海报脸"，刚决定加锁，需后续抽查复现率确认锁是否生效。
2. **封面钩子偏题倾向**：封面 Prompt 偶有偏题/跳题倾向，需观察是否需要再约束。
3. **10 岁身高逐页漂移**：模型同 seed 下仍有逐页身高漂移，纯文字锁压不住，待后续方案（参考锚/seed 策略）。
4. **画风纯净度长期校准**：用户对"通透/明亮/纯净/不脏不灰"诉求强，新书出图后仍需逐页确认未回灰回闷。
