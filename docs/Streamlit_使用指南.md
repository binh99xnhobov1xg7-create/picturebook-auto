## 2. 登录账号

管理员在 Streamlit Cloud → **Settings → Secrets** 配置 `[APP_USERS]`（多用户）或 `APP_USER` + `APP_PASSWORD`（单用户）。

- 登录页填写 **用户名 + 密码**（向管理员索取，**区分大小写**）。
- 登录成功后浏览器会保存 HMAC 令牌（**不含明文密码**）。
- Secrets 结构说明见 [`docs/streamlit-cloud-secrets.md`](streamlit-cloud-secrets.md)。

## 4. 三种制作模式

在 **开始制作** 页顶部选择：

| 模式 | 适用场景 | 产出 |
|------|----------|------|
| **📖 单本制作** | 逐本精调：填故事 → 锁 IP → 出图 → 4 件套 | 绘本 PPT + Worksheet + RR + TG + ZIP |
| **📤 已有绘本 · 教辅三件套** | 已有成品 PDF/PPTX/散图，**不重出绘本** | WS + RR + TG + 抽取摘要（支持单本 / 批量） |
| **📚 批量生产** | 一次粘贴多本大纲，全自动出图（管理员/系列备课） | 每本 × 4 件套；图标记「待人工抽查」 |

## 5. 单本制作：四步向导

AI 抽取完成后，在 **绘本** 角标下按四步操作（顶部有进度条，可点击已解锁步骤回看）：

### Step 1 · 输入 + AI 抽取（表单区）

**必填：**

- **Book Title**（书名）
- **Level**（Smart / 0–6）

**选填（不填则 AI 自动推断）：**

- Book #、Theme、CEFR、Reader Type（L3–6）
- **IP 年龄**：按 Level 自动焊死（L0–2=8 岁、L3–4=10 岁、L5–6=12 岁），一般勿改

**故事：**

1. 填书名 + Level → 点 **「✨ AI 生成故事草稿」**（约 30 秒）
2. 在文本框微调（约 7 句英文）
3. 点 **「🤖 AI 抽取 / 重新抽取」**（约 **2–3 分钟**，**期间勿刷新或关闭页面**）

抽取后共享数据（词表 / 语法 / 拼读 / 题目）会同时供 4 件套使用。

### Step 2 · IP + 画风锁定

1. **🎭 人物 IP**：审核 AI 识别的主角；勾选本书会出现的 IP；映射无名 girl/boy
2. **💇 Mia 发型锁定**（新增）：
   - 默认 **half-up 半扎散发**
   - 可切换 **后脑中高位马尾 ponytail**（对齐官方定妆表）
   - 切换后自动更新分页 Prompt 中的 Mia 参考图
3. **🎨 画风 / 背景**：默认全自动；一般只需确认，高级可展开覆盖
4. 点 **「✅ 人物 + 画风都锁定，进入生图工作台」**

### Step 3 · 生图工作台

- **一键出 8 图**（封面 + 7 页）
- 每页旁显示故事原文；不满意 → **🔁 重生本页**
- 满意 → 勾选 **🔒 锁定**（批量重生时跳过已锁图）
- **🚨 问题反馈面板**：勾选常见问题（跳帧、多指、比例错等）→ 保存并重生，负向词自动注入
- **📝 完整 Prompt 导出**：`image_prompts.txt` + SOP 四部分 + ZIP（无需先出图也可生成）
- 展开 **「分页提示词总览」** 可批量查看 / Claude 润色（一般不用动）
- 全部满意 → **「✅ 图都满意，进入组装」**

### Step 4 · 组装 4 件套

- 切换顶栏 **绘本 / 练习 / RR / TG** 角标可分别微调题目
- 点 **Generate All / 组装** → 下载 **ZIP**（含 4 文档 + 页图）

## 6. 已有绘本 · 教辅三件套

路径：**开始制作** → 选 **「📤 已有绘本 · 教辅三件套」**

### 6.1 单本

1. 填 **Book Title**、**Level**
2. 上传 **PDF / PPTX / 多张图片**（最多 8 张：封面 + 7 页）
3. 粘贴 **故事英文原文**（强烈建议，用于阅读题）
4. 点 **「🚀 生成教辅三件套」**
5. 下载 ZIP（WS + RR + TG + 抽取摘要 + 页图）

> 此路径 **不生成** 出图 Prompt（绘本已存在）。

### 6.2 批量

切到 **「📚 批量」** 子标签：

1. 粘贴多本书信息，**每本用 `===` 分隔**
2. 每本第一行：`Title | Level | Book#`，其后为故事英文原文
3. 上传 **ZIP**（每本一个子文件夹）或 **多个 PDF/PPTX**
4. 选择匹配方式：按书名模糊匹配 / 按列表顺序一一对应
5. 点 **「🚀 批量生成教辅三件套」**
6. 下载 **批量总 ZIP** 或各本 **Teaching Extract**

**格式示例：**

```
How Our Local Government Works | 4 | 18
Mia and Tommy visit city hall. ...
===
Five Ways to Eat Better | 4 | 21
Tommy wants to eat healthier. ...
```

## 7. 批量生产（可选 · 管理员）

- 粘贴多本大纲，**每本用 `===` 分隔**
- 每本第一行：`Title | Level | Book#`，其后为故事原文
- 全自动出图 + 4 件套；完成后可展开 **「批量 Prompt 下载」**

## 8. Prompt 导出说明

| 场景 | 可下载内容 |
|------|------------|
| **单本制作 · 生图工作台** | `image_prompts.txt`、`*_SOP_Prompts.txt`（四部分）、Prompt ZIP |
| **批量生产** | 每本同上（生产时自动落盘） |
| **已有绘本 · 教辅** | 仅 `*_Teaching_Extract.txt`（词表 · 题目摘要），**无** 出图 Prompt |

单页 Prompt 可在工作台展开「分页提示词总览」逐页查看；整本导出用 **「完整 Prompt 导出」** 面板。

## 9. 耐心与防翻车提示

- **AI 抽取** 约 2–3 分钟；**单页生图** 约 20–60 秒；**整本 8 图** 视并发约数分钟。
- 任务进行中 **请勿刷新、关闭标签页或重复点击**，否则会中断。
- 网络慢时先 **Ctrl+F5**；仍异常请联系管理员检查 Cloud Secrets / API Key。
- 无 API Key 时可用 **mock 占位图** 调试流程（设置页可查看连接状态）。
- 正式演示前建议先跑通一本并留一份成品备用。

## 10. 问题反馈清单（生图工作台）

每页图下方 **「🚨 这张图有问题？」** 可勾选：

- 人物跳帧 / 服装发型跳变
- 手指畸形
- 人物过大过小
- 背景杂乱
- 角色比例错
- 视线混乱
- 服装与 IP 不符
- 表情与情绪不符
- 多余或缺失角色

勾选后点 **「保存并重生本页」**，问题描述会自动写入反向 Prompt。

## 11. 常见问题

**Q：刷新后要重新登录？**  
A：检查浏览器是否禁用 localStorage；Cloud 部署后需等 2–5 分钟生效。

**Q：哪些是必填？**  
A：Book Title + Level；故事可 AI 生成。

**Q：4 件套是什么？**  
A：绘本 PPT · Worksheet · Reading Report · Teacher's Guide，共享同一份 AI 抽取数据。

**Q：内网 8501 还能用吗？**  
A：同事日常使用请打开 **Streamlit Cloud 网址**（上文 §1）。本地 8501 仅供开发调试。
